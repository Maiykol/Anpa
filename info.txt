######## How to create a new git ########
 
1. Create new project-folder 
2. Go into the project-folder
3. $ git init
4. (add some files)
5. $ git add . //add all files to git
6. $ git commit -m 'Initial commit'
7. $ git remote add origin <git-url>
8. $ git remote -v //get status of git
9. $ git push --force --set-upstream origin master


######## Get whole git ########

1. Go to folder where you want to place the git
2. $ git clone <git-url>


######## Get part of git ########

1. Create new project-folder 
2. Go into the project-folder
3. $ git init
4. $ git remote add -f origin <git-url>
5. $ git config core.sparseCheckout true
6. Write paths you want to get in file .git/info/sparse-checkout:
    1. $ echo "some/dir/" >> .git/info/sparse-checkout
    2. $ echo "another/sub/tree" >> .git/info/sparse-checkout
7. $ git pull origin master


######## Upload single file to git ########

1. Move file to local git
2. $ git add . //stage file for commit
3. $ git commit -m „adding file XY“ 
4. $ git push origin <branch> 


######## Update whole branch from git ########

1. git pull origin [branchname]



—— USE ONLINE GIT-INTERFACE FOR BRANCHES ——
######## Branches ########

1. Create new branch: $ git checkout -b [branch]
2. Change to branch: $ git checkout [branch]
3. Push branch to GitHub: $ git push [branch]
4. Show all branches: $ git branch
5. Add new remote to branch: $ git remote add [remotename] [git-url]
6. Push changes from committed branch: $ git push [remotename] [branchname]
7. Update branch from repository: 
    1. $ git fetch [remotename]
    2. $ git merge [remotename]/develop
8. Delete local branch: $ git branch -d [branchname] //Force by capital D
9. Delete online branch: $ git push origin :[branchname]
